# Bighonda Studios

Production-ready portfolio website for Saul Loubassa Bighonda, built with Next.js App Router, TypeScript, Tailwind CSS, and editable JSON content files.

## What is included

- Homepage for Bighonda Studios
- Books page and dynamic book detail pages
- Podcast page for Be Ye Transformed
- Purchasable photography gallery with filters and detail modal
- Teaching & Events page with booking form
- About and Contact pages
- SEO metadata, Open Graph, sitemap, robots.txt, and JSON-LD
- Placeholder images in `public/images`
- Optional Decap CMS files in `public/admin`

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run lint
npm run build
```

## Edit content

Most copy and links live in the JSON files under `content/`. Start with `EDITING_GUIDE.md`.
